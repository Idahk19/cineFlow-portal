# CINEFLOW PORTAL

This is a web based form designed to onboard new content creators into the Cineflow Platform

## Overview

The project focuses on collecting creator information in a structured, user-friendly, and responsiveinterface. The webpage focuses on simplicity, usability and responsiveness while showcasing multimedia element such as a welcome video

## Visual Identity
- Images: Clean and modern visuals used for branding and onboarding experience
- Video: Embedded welcome video introducing the CineFlow platform
- Typography: Clean fonts for readability and professionalism
- Layout: Structured form sections for better user experience

## Key Features

### Semantic HTML Structure

The project uses proper semantic HTML elements such as:

- header
- main
- section
- form
- fieldset

This improves:

- Accessibility
- SEO optimization
- Code readability

### Interactive Registration Form

The form collects essential creator details including:

- Full Name
- Email Address
- Date of Birth
- Username
- Password
- Profile Picture Upload
- Content Preferences

### Form Validation

The form includes built-in HTML validation:

- Required fields
- Email format validation
- Pattern validation for username
- File type restriction for uploads

### User Experience Design

- Responsive layout using modern CSS utility classes
- Clear form grouping using fieldsets
- Improved spacing and structure using - Tailwind CSS
- Interactive inputs

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)
- Media elements (video, image, file upload)
